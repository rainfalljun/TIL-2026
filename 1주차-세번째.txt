클라우드 수업 깃 설명 정리
Git 명령어 모음집
Git 주요 명령어 모음

1.  설정 및 시작

  - git init: 현재 폴더에서 Git 저장소 시작
  - git config --global user.name "이름": 사용자 이름 설정
  - git config --global user.email "이메일": 사용자 이메일 설정

2.  파일 추가 및 저장 (가장 많이 사용)

  - git status: 현재 저장소의 파일 상태 확인
  - git add 파일명: 특정 파일을 스테이지(장바구니)에 추가
  - git add .: 변경된 모든 파일을 스테이지에 추가
  - git commit -m "메시지": 스테이지의 파일을 기록(포장)으로 남김

3.  원격 저장소 (GitHub) 연결

  - git remote add origin 주소: 깃허브 저장소 연결
  - git push origin main: 로컬 기록을 깃허브로 업로드
  - git pull origin main: 깃허브의 내용을 내 컴퓨터로 가져오기
  - git clone 주소: 깃허브 저장소를 내 컴퓨터로 그대로 복제

4.  확인 및 기타

  - git log: 지금까지 만든 커밋 기록 확인
  - ls: 현재 폴더 파일 목록 보기
  - pwd: 현재 폴더 경로 확인
  - cd 폴더명: 특정 폴더로 이동
  - cd ..: 상위 폴더로 이동
