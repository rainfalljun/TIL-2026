  • messages — content · location(GEOGRAPHY) · altitude · category · echo_count · expires_at(+24h) · parent_id
  • echos — UNIQUE(message_id, user_id) + INSERT 트리거 → expires_at 연장 + echo_count +1
  • safe_zones — user_id · location · radius_m
익명 인증 흐름
  • signInAnonymously() → 가입 없이 UUID 발급
  • RLS: 인증 사용자만 INSERT 허용
  • _requireUserId() 패턴 — 세션 만료 시 자동 복구
get_nearby_messages RPC
WHERE ST_DWithin(
  m.location,
  ST_SetSRID(ST_MakePoint(
    lng, lat), 4326)::GEOGRAPHY,
  radius_m -- 기본 500m
)
AND m.reporter_count < 5
AND m.expires_at > NOW()
AND m.parent_id IS NULL
-- 루트 메시지만 반환 (답장 제외)
ORDER BY m.created_at DESC
