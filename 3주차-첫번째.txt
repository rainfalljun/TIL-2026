CLAUDE.md란?

claude.md는 인공지능 모델인 클로드(Claude)가 특정 프로젝트나 코딩 작업을 수행할 때 지켜야 할 지침을 담은 설정 파일입니다.

주로 개발자들이 자신의 프로젝트 폴더 최상단에 이 파일을 만들어 두는데, AI가 코드를 작성하거나 수정할 때 사용자의 코딩 스타일이나 프로젝트의 고유한 규칙을 잊지 않도록 돕는 역할을 합니다.

이 파일을 활용하는 구체적인 방법과 단계는 다음과 같습니다.

1. 파일 만들기
먼저 프로젝트 폴더 안에 새로운 파일을 하나 생성해야 합니다.

사용 중인 코드 편집기(예: VS Code)를 엽니다.

프로젝트의 가장 바깥쪽 폴더(최상위 루트)를 마우스 오른쪽 버튼으로 클릭합니다.

새 파일 만들기(New File)를 누릅니다.

파일 이름을 claude.md라고 입력하고 엔터를 칩니다.

2. 내용 작성하기
파일을 만들었다면 이제 클로드가 읽을 내용을 적어줍니다. 복잡한 프로그래밍 언어가 아니라 일반적인 글로 적어도 괜찮습니다.

파일의 맨 윗부분에 이 프로젝트가 무엇인지 간단히 적습니다.

클로드가 코드를 짤 때 꼭 지켰으면 하는 규칙을 번호를 매겨서 작성합니다.

예를 들어 "변수 이름은 영어로 지어줘"라거나 "코드를 수정할 때마다 주석을 달아줘" 같은 내용을 적으면 됩니다.

3. 클로드에게 알려주기
파일을 다 만들었다면 클로드에게 이 내용을 참고하라고 말해주어야 합니다.

클로드와 대화를 시작하는 창으로 갑니다.

파일 첨부 버튼을 누릅니다.

방금 만든 claude.md 파일을 선택해서 올립니다.

"이 파일에 적힌 지침대로 내 프로젝트를 도와줘"라고 입력합니다.
