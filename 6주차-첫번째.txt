
6주차 API 연동 및 냉장고를 부탁해 앱 제작 요약

1. API 개념 및 보안
- API는 프로그램과 AI 모델을 이어주는 다리 역할을 하며 외부의 강력한 기능을 내 앱에 추가할 수 있게 합니다.
- API 키는 인증을 위한 비밀 열쇠이므로 코드에 직접 쓰지 않고 .env 파일에 저장해 관리해야 합니다.
- .gitignore 파일은 API 키가 들어있는 .env 파일이 깃허브 등에 공유되지 않도록 차단하는 역할을 합니다.

2. 사용 AI 모델 (2026년 4월 기준)
- 이미지 인식: 냉장고 사진 속의 재료를 분석하기 위해 Gemma 3 27B 무료 모델을 사용합니다.
- 텍스트 생성: 인식된 재료로 레시피를 만들기 위해 Qwen 3.6 Plus 무료 모델을 사용합니다.
- 오픈라우터 플랫폼을 이용하면 여러 모델을 하나의 키로 통합 관리할 수 있어 편리합니다.

3. 앱 개발 3단계 전략
- 1단계: Streamlit을 이용해 웹 화면을 만들고 사진에서 재료를 인식하는 기능을 구현합니다.
- 2단계: 인식된 재료를 바탕으로 레시피를 생성하고 사용자가 재료를 편집할 수 있게 합니다.
- 3단계: 로그인 기능과 함께 마음에 드는 레시피를 저장할 수 있는 대시보드를 추가합니다.

4. 주요 도구 및 주의사항
- Streamlit은 파이썬만으로 AI 시연용 웹 앱을 빠르게 만들 수 있게 도와주는 도구입니다.
- 실행 시 모델명에 오타가 있거나 .env 파일이 작업 폴더에 없으면 오류가 발생할 수 있으니 주의해야 합니다.
